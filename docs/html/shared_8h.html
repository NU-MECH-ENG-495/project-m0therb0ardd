Loads and scales an image to the given matrix dimensions.

Uses GraphicsMagick to load a static image or first frame of a GIF, and resizes it to the target dimensions (e.g., 64x64) for display on the LED matrix.

Parameters
filenamePath to the image file to load.
target_widthDesired output width in pixels.
target_heightDesired output height in pixels.
Returns
ImageVector Vector of Magick::Image objects. Returns an empty vector on error.

< Loads frames from image file

< Use the first frame only (good for static or GIF)

< Resize image to match matrix
